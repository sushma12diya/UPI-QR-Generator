# UPI-QR-Generator
Built a tool to generate dynamic payment QR codes compatible with PhonePe, Google Pay, and Paytm.
Automated QR creation from user UPI IDs and transaction data using the qrcode library.
Used Pillow (PIL) to process, display, and save QR images locally for merchant use.
Programmed custom payment URLs with Recipient Name (PN) and UPI ID (PA) parameters for seamless routing.
